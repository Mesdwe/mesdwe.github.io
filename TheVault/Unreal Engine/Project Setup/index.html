需要下载：

  1. P4V (有管理权限需求可另外下载P4Admin）
  2. UE 5.5
  3. Visual Studio 2022 (或其他版本) 需选择安装:
    • Desktop development with C++
    • Game development with C++
    • .NET desktop development
    • Universal Windows Platform development 具体版本&推荐配置参照 这里

Perforce 步骤

  1. 运行P4V
  2. 需要填写：
    1. Server: ec2-35-179-105-89.eu-west-2.compute.amazonaws.com:1666
    2. User: damien, hailin, vesper, zico, art(doro和提纲共用)，密码均为password，邮箱与用户名相同![[Pasted image 20250303194352.png]]
    3. Workspace选择New，给你的Workspace起个名，Workspace root选择你要工作的本地文件目录；Stream选择main![[Pasted image 20250305145853.png]]
  3. 进入后右键根目录选择Get Latest Revision ![[Pasted image 20250305151447.png]]

UE project setup

  1. 打开RoadTrip.uproject。第一次打开需要build，提前安装好visual studio ![[Pasted image 20250305213106.png]]
  2. 成功打开后需在引擎内connect version control。编辑器右下角选择Revision Control -> Connect to Revision Control -> Provider选择Perforce, 填写ec2-35-179-105-89.eu-west-2.compute.amazonaws.com:1666和用户名即可![[Pasted image 20250305213645.png]]
  3. 完成后显示绿色对钩即为成功 ![[Pasted image 20250305214434.png]]
